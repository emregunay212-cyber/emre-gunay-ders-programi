# Emre Günay – Ders Programı

Bilnet Balıkesir Bilişim Teknolojileri öğretmeni Emre Günay'ın haftalık ders programını gösteren, mobil uyumlu, tek sayfalık statik web sitesi.

## Özellikler

- Anlık saat ve tarih
- Şu anki ders (sınıf, tür, kalan dakika, progress bar)
- Sonraki ders bilgisi
- Haftalık program özeti (renk kodlu)
- Dark tema, mobil öncelikli tasarım
- Saf HTML/CSS/JS — framework yok

## Kullanım

`index.html` dosyasını tarayıcıda açmak yeterli.

## Deploy

Vercel'e sürükle bırak ile yayınlanır:

1. [vercel.com/new](https://vercel.com/new) adresine git
2. `index.html` dosyasını sürükle bırak
3. Otomatik yayınlanır
